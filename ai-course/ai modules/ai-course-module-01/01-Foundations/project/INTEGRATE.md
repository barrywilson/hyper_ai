# Integrate learning with reflection

Take 20–30 minutes to write up your reflections before our next session. Be specific — the more concrete you are, the more useful this is as a reference later.

1. Walk through the spec → prompt → plan → implement cycle you just ran. Where did you feel in control? Where did things go sideways?
2. What did you learn about how much detail your assistant needs? Did more detail in the spec produce better results, or did it add noise?
3. Did the journal feel useful or like overhead? What would make it more valuable to you?
4. What surprised you about not writing the code directly — either how that felt, or what the output actually looked like?
5. What rules did you end up capturing? What behaviour made you want to write them down?
6. What would you do differently on the next run?

Before our next session, post a link to your repo and your reflections in the group chat.
