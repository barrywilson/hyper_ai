# Agent Instructions

This project has a context framework in `memory/`. Read these files at the start of each conversation:

- `memory/ABOUT.md` — project purpose, personas, and constraints
- `memory/ARCHITECTURE.md` — system design, patterns, and key decisions
- `memory/IMPLEMENTATION.md` — tech stack, languages, dependencies, and preferences

Load all three before asking any questions about the project. If a task only touches one area, focus on the relevant file — but when in doubt, read all three first.

When you discover something about the project that isn't captured here, update the relevant file.
