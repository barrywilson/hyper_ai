# Integrate learning with reflection

Take 20–30 minutes to write up your reflections before our next session.

1. Did the context framework change the quality of your assistant's output? Can you point to a specific example where it made a noticeable difference?
2. What was harder: defining context upfront or keeping it accurate as the project evolved?
3. Did you find yourself updating the context documents during the exercise? What triggered those updates?
4. Which context document felt most valuable to your assistant — and which felt hardest to write? Why?
5. What surprised you about how the assistant used (or didn't use) the context you provided?
6. How would you approach building a context framework for a project you're actually working on?

Before our next session, post a link to your repo and your reflections in the group chat.
